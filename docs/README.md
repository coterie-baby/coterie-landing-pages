## Product Summary

Coterie Landing Pages is a Next.js-based marketing platform that serves as the primary customer acquisition funnel for Coterie's premium baby care products. The platform features:

- **Interactive Sizing Quiz** - Guides parents to personalized diaper size recommendations
- **Dynamic Marketing Pages** - CMS-managed landing pages for campaigns
- **Multi-channel Attribution** - Comprehensive tracking across ad platforms
- **Mobile-First Experience** - Optimized for mobile traffic from paid campaigns

## Key Metrics & KPIs

The platform is instrumented to track:

- Quiz completion rate
- Quiz-to-cart conversion
- Page engagement metrics
- Attribution by marketing channel
- Cross-domain user journey
